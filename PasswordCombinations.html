Here we are supposing two variables,
x and y , where,


x = Number of Digits
y = Number of Blanks


If:
x ≥ y


The formula will be: x y

For example:
If: Number of Digits = 10

Number of Blanks = 4

Then:

Total Number Of Password Combinations
= 104
= 100000




If:
x < y


The formula will be: x x

NOTE:- In this case, we are not considering the BLANK SPACES as Elememts to fill in the blanks.

For Example: Suppose that we have:-

NUMBER OF DIGITS = 3
NUMBER OF BLANKS = 4

Then our password could be:

111; 112; 113; 121; 122; 123; 131; 132; 133; 211; 212; 213; 221; 222; 223; 231; 232; 233; 311; 312; 313; 321; 322; 323; 331; 332; 333. (Total = 27)

But it can't be:

11_1; 11_2; 11_3; 1_11; 1_12; 1_13; _111; _112; _113, etc. (Total = 256)


For Example:
If: Number Of Digits = 4
Number Of Blanks = 6

Then:

Total Number Of Password Combinations
= 44
= 256




If:
x = 0
y = 0


Then formula could be: x * y

As in all of these 3 cases, it is
NOT POSSIBLE to have any Password.


For Example:
If: Number Of Digits = 0
Number Of Blanks = 4

Then:

Total Number Of Password Combinations
= 0 * 4
= 0
